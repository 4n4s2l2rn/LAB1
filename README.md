# HelloToast 

Une application Android simple et intuitive pour apprendre les bases du développement mobile, notamment la gestion des événements de clic, l'affichage de messages (Toasts) et la manipulation des composants d'interface (UI).

## Aperçu
L'application se compose d'une interface épurée avec deux boutons principaux :
* **Toast** : Affiche un message éphémère "Bonjour !" en bas de l'écran.
* **Count** : Incrémente un compteur affiché dynamiquement au centre.

## Fonctionnalités
  Affichage de notifications **Toast**.
  Gestion d'un état de compteur en temps réel.
  Mise en page adaptative avec `LinearLayout`.
  Utilisation des expressions Lambda pour les écouteurs de clics (`setOnClickListener`).

## Technologies utilisées
- **Langage** : Java
- **IDE** : Android Studio
- **Layout** : XML (LinearLayout)
- **Minimum SDK** : API 24 (Android 7.0)

## Installation

1.  Clonez ce dépôt :
    
    git clone [https://github.com/4n4s2l2rn/LAB1.git](https://github.com/4n4s2l2rn/LAB1.git)
  
2.  Ouvrez le projet dans **Android Studio**.
3.  Laissez Gradle synchroniser les dépendances.
4.  Lancez l'application sur un émulateur ou un appareil physique.

##Structure du projet
* `MainActivity.java` : Contient la logique de programmation pour les boutons et le compteur.
* `activity_main.xml` : Définit l'interface utilisateur (boutons et texte).
* `AndroidManifest.xml` : Déclare les composants de l'application.
